# 📂 ZipTree - Connected Project Tree Extractor

**ZipTree** is a professional, high-performance developer tool designed to extract and visualize the full folder structures and file hierarchies of compressed `.zip` archives. It generates clean, flawless, and perfectly connected text-based folder trees (`├──` and `└──`) that are ready to be used in documentations, GitHub READMEs, or project breakdowns.

This tool is optimized, redesigned, and maintained under the official **ShiftDev** brand.

---

## ✨ Features

- **Drag & Drop Interface:** Smooth and highly interactive UI built with modern CSS variables.
- **Flawless Connector Logic:** Fixes typical broken vertical paths (`│`) to ensure 100% connected trees, even in deeply nested directory levels.
- **Client-Side Processing:** Files are processed entirely inside the browser using `JSZip` — super fast and completely secure (No data is uploaded to any servers).
- **One-Click Download:** Export your generated project schema directly as a clean `.txt` file instantly.
- **Fully Responsive Dashboard:** Tailored using modern aesthetics, developer-friendly monospace typography (`Fira Code`), and clear accessibility traits.

---

## 🚀 How to Use

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. Drag and drop any `.zip` archive into the dashed zone, or click to browse.
4. Preview the structured text layout instantly on your dashboard.
5. Click **"تحميل الشجرة كـ ملف نصي (TXT)"** to download the structure output.
